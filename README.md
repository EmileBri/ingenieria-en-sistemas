# Ingeniería en Sistemas Computacionales — ISC, TecNM Campus Victoria

Proyectos seleccionados de la licenciatura en Ingeniería en Sistemas Computacionales (Instituto Tecnológico de Ciudad Victoria, TecNM), previos a la maestría MCID en la FIC-UAT.

## Contenido

| Carpeta | Materia | Descripción |
|---------|---------|-------------|
| `taller_base_de_datos/` | Taller de Base de Datos, 5° semestre | Consultas SQL, comandos DML y procedimientos almacenados |
| `graficacion/transformaciones_2d/` | Graficación, 5° semestre | App Java con transformaciones geométricas 2D |
| `graficacion/transformaciones_3d/` | Graficación, 5° semestre | App Java con transformaciones geométricas 3D |

## Taller de Base de Datos

Tres scripts SQL sobre una base de datos académica:

- `consultas.sql` — consultas SELECT con JOINs y agrupaciones
- `comandos_dml.sql` — operaciones INSERT, UPDATE y DELETE
- `procedimientos_almacenados.sql` — procedimientos con parámetros de entrada y salida

## Graficación

Dos aplicaciones Java con interfaz gráfica (Swing/AWT) que implementan transformaciones geométricas mediante matrices:

- **2D**: traslación, rotación, escalado y sus versiones genéricas e inversas sobre figuras en el plano
- **3D**: las mismas transformaciones extendidas al espacio tridimensional con manejo de color

## Tecnologías

Java, Swing/AWT, SQL (MySQL)
